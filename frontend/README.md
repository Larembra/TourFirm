# Турфирма — демо-фронтенд

React-приложение для работников туристической фирмы.

## Что уже есть

- вкладки: `Сведения о клиентах`, `Сведения о путевках`, `Сведения о продаже путевок`, `Личный кабинет`;
- стартовая страница для гостя со справочной информацией;
- заглушка авторизации/регистрации — любые введённые данные входят в тестового пользователя-руководителя;
- списки клиентов, путёвок и продаж на статических данных;
- отчёты: горящие путёвки, самая дорогая путёвка, потери от скидок, популярные путёвки;
- базовые действия менеджера и расширенные действия руководителя.

## Как запустить

В папке `frontend` выполните:

```powershell
npm install
npm start
```

После запуска откройте браузер по адресу:

```text
http://localhost:3000
```

## Тестовый сценарий

1. Откройте главную страницу.
2. Нажмите `Авторизоваться`.
3. Введите любые значения в поля формы и нажмите `Войти как руководитель`.
4. Откроется личный кабинет тестового руководителя.
