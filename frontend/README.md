# Tourfirm — фронтенд

React приложение (Create React App) для интерфейса сотрудников турфирмы.

Особенности
- Страницы: главная, клиенты, путёвки, продажи, управление менеджерами, отчеты, личный кабинет.
- Модальные формы для создания/редактирования путёвок с загрузкой изображений.
- Использует API бэкенда по умолчанию на http://127.0.0.1:8000

Запуск локально
1. Установите зависимости и запустите dev-сервер:

```powershell
cd frontend
npm install
npm start
```

2. Откройте http://localhost:3000


Сборка
- Создать production-бандл:

```powershell
npm run build
```
